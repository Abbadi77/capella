Design > Coverage


DCOV_01 - Interface used one component at least
This rule checks that an Interface is used by at least one Component.


DCOV_02 - Capability exploitation check
This rule checks that a Capability is exploited by at least one Mission.


DCOV_03 - Mission capability exploitation test
This rule checks that a Mission exploit at least one Capability.


DCOV_04 - Mission involves an actor check
This rule checks that a Mission involves at least one actor.


DCOV_05 - System Component involvement in a mission check
This rule checks that an System Component is involved in at least one Mission.


DCOV_06 - System Component participation in capabilities minimal number
This rule checks that the minimum number of participation from a Component towards a Capability is one.


DCOV_07 - Capability interation with actors
This rule checks that a Capability is involved by at least one Actor.


DCOV_08 - System Component involvement in at least one scenario related to the capabilities it participates
This rule checks that the current Component involved with capabilities appears in at least one scenario related to the capabilities it participates.


DCOV_09 - Component port Component Exchanges check
This rule generates a warning if there is neither Component Exchanges using the target Component Port nor providing/requiring any Interface.


DCOV_10 - Function port exchanges check
This rule generates a warning if there is no functional exchange using the target Input/Output Function Port


DCOV_11 - Physical port Physical Link check
This rule generates a warning if there is no Physical Links using the target Physical Port.
